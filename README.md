# 野生工大助手后端

2020中国高校计算机大赛 微信小程序应用开发赛 决赛入围后被淘汰的垃圾之作

## 配置

+ 需要在 `config/Cache.php` 实现一个缓存类
+ 需要在 `config/Storage.php` 实现一个存储类
+ 需要配置 `config/Settings.php`
